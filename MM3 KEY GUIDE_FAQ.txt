===============================================================================
[SEC1] EXPLORATION & OVERWORLD
===============================================================================

* MOVE:          D-Pad (or LT + Joystick). 
* SIDESTEP:      LT + RT + Joystick.
* INTERACT:      RT (Maps to SPACE).
* MOUSE MODE:    Left Joystick. Hold X for Left Click.
* SHOOT/BASH:    LT + D-UP. (Click to Shoot / Hold to Bash).
* REST/DISMISS:  LT + D-DOWN. (Click to Rest / Hold to Dismiss).

===============================================================================
[SEC2] COMBAT & QUICK FIGHT
===============================================================================

* STANDARD:      A (Attack), B (Quick Fight), C (Cast).
* QUICK FIGHT:   Executes a pre-set action (Attack/Block/Cast/Retreat).
    - CYCLE:     Hold B + D-RIGHT to rotate stances.
    - LOCK/SET:  Hold B + D-DOWN or press START.
* MODIFIERS:     D-UP + A (Block) | D-DOWN + A (Retreat).

===============================================================================
[SEC3] INVENTORY & PARTY MANAGEMENT
-------------------------------------------------------------------------------
  Note: D-PAD Cross-Logic is active while holding RT.
-------------------------------------------------------------------------------

* SELECT CHAR:   LT + [X, Y, Z / C, B, A / START, SELECT].
* OPEN BAG:      LT + Hold Character Button (or LT + L3).
* IN-BAG (RT):   While holding RT, use D-PAD:
    - USE ITEM:  Tap D-PAD UP.
    - DISCARD:   Tap D-PAD DOWN.
    - EQUIP:     Tap D-PAD RIGHT.
    - REMOVE:    Tap D-PAD LEFT.
* TRANSFER:      While holding RT, click the recipient's button.
                 *PRO: Hold the button to jump directly into their inventory.*

===============================================================================
[SEC4] SPELL MENU & ELEMENTAL ARROWS
===============================================================================

* OPEN MENU:     C (Click for Current / Hold for New).
* SHIFT (C-HOLD):D-UP: Fire | D-RIGHT: Elec | D-DOWN: Cold | D-LEFT: Acid.
* WIZARD EYE:    B + Y (Quick Cast).
* UTILITY MACRO: B + D-DOWN (Casts Wizard Eye, Levitate, Light, Walk on Water).
* BUFF MACROS:   B + D-LEFT (Prot Elements) | B + D-UP (Exploration Buffs).

===============================================================================
[SEC5] QUICK SPELLS & TRANSPORT MACROS
-------------------------------------------------------------------------------
  Input Logic: A + D-PAD
-------------------------------------------------------------------------------
* D-LEFT:        Transport Box.
* D-UP:          Teleport.
* D-DOWN:        Town Portal.
* D-RIGHT:       Lloyd's Beacon (Cast).
    - SET:       When prompted, hold D-LEFT.
    - RETURN:    When prompted, hold D-RIGHT.

* ENTER NUMS:    Hold RT and use:
                 [X:1] [Y:2] [Z:3] [C:4] [B:5] [A:6] [ST:7] [SL:8] [J-E:9] [J-W:0]

===============================================================================
[SEC6] MAGIC MIRRORS & SYSTEM
===============================================================================

* MIRRORS:       L3 + [Z:Home] [Y:Seadog] [X:Freeman] [C:Doomed] [B:Redhot] [A:Arena]
                 *Use D-Pad for Elemental Planes.*
* RA MENU:       Hold START + SELECT.

===============================================================================
[SEC7] SHOPS & SERVICES
-------------------------------------------------------------------------------
  Logic: Directions correspond to visual list order (1st, 2nd, 3rd Choice).
-------------------------------------------------------------------------------

* BLACKSMITH / TEMPLE (Y): Hold Y + Direction.
    - D-UP:      Buy / Heal / Train (1st).
    - D-RIGHT:   Armor / Donate (2nd).
    - D-DOWN:    Sell / Uncurse (3rd).

    * NESTED BUY MENU:  D-UP: Weapons | D-RIGHT: Armor | D-DOWN: Misc.
    * NESTED SELL MENU: D-UP: Sell | D-RIGHT: Repair | D-DOWN: Identify.

* TAVERN / GUILD (Z): Hold Z + Direction.
    - D-UP:      Drink / Info (1st).
    - D-RIGHT:   Food / Buy Spells (2nd).
    - D-DOWN:    Tip / Spell Info (3rd).
    - D-LEFT:    Rumor (4th).

* BANK (X): Hold X + Direction.
    - D-UP/DN:   Gold / Gems.
    - SELECT:    D-LEFT (Deposit) | D-RIGHT (Withdraw).

===============================================================================
[SEC8] CHARACTER CREATION & INN
===============================================================================

* NAVIGATE:      D-UP / D-DOWN.
* CYCLE OPTIONS: Hold SELECT and tap LT or RT.
* REROLL STATS:  Hold SELECT and press X.
* SAVE/CONFIRM:  Hold START (Enter).
* EXIT/DISCARD:  Hold SELECT (Esc).

===============================================================================
[SEC9] ON-SCREEN KEYBOARD (OSK)
===============================================================================

* ACTIVATE:      Click Left Joystick (L3).
* TYPING:        D-Pad (Move) | A (Select) | X (Backspace).
* MODIFIERS:     Y (Space) | START (Enter).
* EXIT:          Click L3 again or press B.

===============================================================================
[SEC10] PRO-TIPS FOR EFFICIENCY
===============================================================================

* GEAR SHUFFLE:  Hold SELECT (Fast Mode) + LT/RT to scroll through party.
                 Tap A to equip gear instantly without menu loops.
* CONFIRMATION:  START is always "Yes." SELECT is always "No/Esc."

===============================================================================
[SEC11] RETROARCH NAVIGATION
-------------------------------------------------------------------------------
  Note: Game pauses while RA Menu is open.
-------------------------------------------------------------------------------

* MOVE:          Hold START + D-PAD UP/DOWN.
* ENTER:         Hold START + D-PAD RIGHT.
* BACK:          Hold START + D-PAD LEFT.

===============================================================================
[SEC12] SCENARIO WALKTHROUGHS
===============================================================================

[A] TRADING GEAR:
    1. LT + Hold X (Character 1's bag opens).
    2. Use D-Pad to highlight the item.
    3. Hold RT + Tap B (Sends item to Character 5). 
    4. Keep holding B to jump directly into Character 5's bag to equip it.

[B] HEALING:
    1. LT + Tap Z (Cleric).
    2. Tap C (Spells) -> Highlight "Heal" -> START.
    3. LT + Tap X (Target Char 1).

[C] BLACKSMITH BUY/SELL:
    1. Y + D-UP (Enter Buy).
    2. Y + D-RIGHT (Select Armor) -> START to buy.
    3. Exit -> Y + D-DOWN (Enter Sell).
    4. Y + D-UP (Choose Sell) -> Hold Y (Confirm Yes).
